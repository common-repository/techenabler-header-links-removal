== Description ==

Hackers frequently base on known vulnerabilities to attack websites. This Header Links Removal plugin secures your website by removing the version of core Wordpress functions as well as unneccessary <link> tags from displaying in the HTML source, thus making it harder for hackers to exploit on vulnerabilities when your Wordpress and plugins are not updated to the latest versions. 

== Installation ==

Install from your WordPress "Plugins > Add New" and search for "TechEnabler Header Links Removal" plugin. 

Follow these steps for manual installation:

1. Upload the zip file and unzip it in the "/wp-content/plugins/" directory.
2. Activate the plugin "TechEnabler Header Links Removal" in the "Plugins" page.
